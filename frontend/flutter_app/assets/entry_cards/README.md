This folder contains the entry-card artwork used on the first screen of the Flutter app.

Files:
- `one_shot.jpg`: artwork for `One-Shot`
- `campagna_corta.jpg`: artwork for `Mini-campagna`
- `campagna_lunga.jpg`: artwork for `Campagna lunga`
- `dungeon.jpg`: artwork for `Esplorazione dungeon`
- `*.svg`: previous placeholder art kept as optional fallback/reference

These are local app assets, so the app works offline and does not depend on third-party image hosts.
If you replace them with custom artwork, keep the same filenames or update the paths in
`lib/src/ui/pages/campaign_builder_page.dart`.
