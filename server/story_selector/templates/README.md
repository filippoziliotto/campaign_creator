# Templates

Template markdown usati dal renderer backend per costruire il prompt finale.

## File
- `prompt_template.md`: fallback generico se il tipo campagna non ha un template dedicato.
- `prompt_template_one_shot.md`: template per avventure in singola sessione.
- `prompt_template_mini_campaign.md`: template per mini-campagne (3-6 sessioni).
- `prompt_template_long_campaign.md`: template per campagne lunghe con archi multipli.
- `prompt_template_dungeon_exploration.md`: template per campagne centrate su esplorazione dungeon.
